# Matrix Infinity Flow – Hacker Glitch Effect

A **Matrix-inspired digital rain animation** combined with a **glitching “HACKER” text effect**, built using **HTML, CSS, and JavaScript**.  
This project recreates the iconic Matrix terminal aesthetic with infinite flowing binary numbers and a centered flicker/glitch display of the word, "HACKER".

## Features

- Infinite Matrix-inspired digital rain animation
- Flickering and glitching text effect
- Centered glowing display
- Full-screen responsive canvas
- Dark hacker theme
